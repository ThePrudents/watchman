watchman
===========

[![Build Status](https://travis-ci.org/ThePrudents/watchman.svg?branch=master)](https://travis-ci.org/ThePrudents/watchman)

Facebook's watchman

Example of usage (all parameters are optional)

Simple

  roles:
    - {
        role: "watchman"
      }


Advanced:


  roles:
    - {
        role: "watchman",
        watchman_version: 4.5.0,
      }
